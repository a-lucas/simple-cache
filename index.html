Redis url cache

An extensible node.js URL caching library

View on Github Documentation Download .tar.gz

redis-url-cache helps you store and retrieve url content on a Redis datastore.

Getting started

Instalation


    npm install redis-url-cache

Configuration

Create a cache-config.js file with the following content


    module.exports = {
        always: [],
        maxAge: [],
        never: []
        default: 'always'
    }

This will store indefinitively all urls by default
